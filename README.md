# ThreatLens AI

ThreatLens AI is an AI-powered SOC/SIEM/SOAR-style security operations platform that simulates real-time threat monitoring, detection, investigation, response, and reporting.

The system generates live JSON security events, stores them in MongoDB, detects suspicious behavior using rule-based detection, creates alerts, groups related alerts into incidents/cases, and uses a multi-agent AI workflow to investigate threats and generate professional SOC reports.

## Key Features

- JWT authentication and role-based access control
- Automatic live JSON security event generation
- Rule-based detection for brute force, suspicious login, data exfiltration, port scans, privilege escalation, and admin activity
- Alerts and incident/case management
- Multi-agent AI investigation workflow
- SOAR-style playbook recommendations with human approval
- Professional SOC report generation
- Futuristic cybersecurity dashboard with live monitoring visuals

## Tech Stack

**Frontend:** React, Vite, Tailwind CSS, Recharts, Three.js / React Three Fiber  
**Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT, bcrypt  
**AI Service:** Python, FastAPI, LLM-based multi-agent workflow  
**Deployment:** Vercel, MongoDB Atlas, GitHub

## Purpose

ThreatLens AI demonstrates full-stack development, cybersecurity detection logic, AI-assisted investigation, safe automation, SOC workflows, database design, and professional dashboard UI/UX in one flagship portfolio project.
